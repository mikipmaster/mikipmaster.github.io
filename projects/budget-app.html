Budget Web Application

Nowoczesna, responsywna aplikacja webowa do zarządzania budżetem domowym. Full-stack rozwiązanie z React.js frontend, Node.js backend, MongoDB bazą danych i bezpieczną autoryzacją JWT. Projekt oferuje kompletny system finansowy z raportami, eksportem danych i zaawansowaną analizą wydatków.

Zobacz kod źródłowy Live Demo

O Projekcie

Budget Web Application to kompleksowe rozwiązanie full-stack stworzone z myślą o osobach, które chcą świadomie kontrolować swoje finanse. Aplikacja odpowiada na realną potrzebę rynku: brak prostych, darmowych i spersonalizowanych rozwiązań finansowych dostępnych z poziomu przeglądarki.

System umożliwia szybkie rejestrowanie przychodów i wydatków, analizę struktury wydatków oraz generowanie czytelnych raportów finansowych z możliwością eksportu danych do CSV.

Technologie Stack

React.js Node.js Express.js MongoDB Mongoose JWT Tailwind CSS React Router Chart.js Hostinger

Kluczowe Funkcjonalności

  • Bezpieczna rejestracja i logowanie (JWT)
  • Dodawanie, edycja i usuwanie transakcji
  • Historia transakcji z kategoryzacją
  • Interaktywne raporty graficzne
  • Dashboard z podsumowaniem finansowym
  • Filtrowanie i sortowanie danych
  • Eksport danych do CSV
  • Responsywny design (mobile-first)
  • Zabezpieczenie tras (ProtectedRoute)
  • Obsługa błędów i walidacja

Architektura i Bezpieczeństwo

Aplikacja została zbudowana w oparciu o zasady SOLID i DRY. Frontend wykorzystuje React Context do zarządzania stanem, podczas gdy backend oferuje RESTful API z middleware do autoryzacji. Bezpieczeństwo zapewniają JWT tokeny, walidacja danych i ochrona przed XSS/CSRF.

Zarządzanie Transakcjami

Intuicyjny system dodawania, edycji i kategoryzacji transakcji. Użytkownicy mogą łatwo rejestrować przychody i wydatki z automatyczną walidacją danych i przypisywaniem do odpowiednich kategorii finansowych.

  • Szybkie dodawanie transakcji
  • Edycja i usuwanie zapisów
  • System kategorii i tagów
  • Filtrowanie po datach
  • Wyszukiwanie transakcji

Raporty i Analiza Danych

Zaawansowany system raportowania z interaktywnymi wykresami wykorzystującymi Chart.js. Aplikacja generuje wykresy słupkowe i kołowe pokazujące strukturę wydatków, trendy finansowe i porównania okresowe.

  • Wykresy słupkowe przychodów/wydatków
  • Analiza kategorii wydatków
  • Trendy finansowe w czasie
  • Raporty miesięczne/roczne
  • Eksport raportów do CSV

Bezpieczna Autoryzacja

Kompleksowy system autoryzacji wykorzystujący JWT (JSON Web Tokens) z hashowaniem haseł, walidacją formularzy i zabezpieczeniem tras. Aplikacja implementuje ochronę przed XSS, CSRF i zapewnia bezpieczne przechowywanie danych użytkowników.

  • JWT Authentication & Authorization
  • Hashowanie haseł (bcrypt)
  • Protected Routes
  • Ochrona przed XSS/CSRF
  • Secure MongoDB integration

Architektura Techniczna

Frontend - React.js

Single Page Application z React Router, Context API do zarządzania stanem, Tailwind CSS do stylowania i responsywnego designu

  • React Hooks (useState, useEffect, useContext)
  • Protected Routes & Navigation Guards
  • Component-based Architecture
  • Mobile-first Responsive Design

Backend - Node.js/Express

RESTful API z Express.js, middleware do autoryzacji, walidacji danych i obsługi błędów

  • Express.js RESTful API
  • JWT Middleware & Auth Guards
  • Data Validation & Sanitization

Database - MongoDB

NoSQL baza danych z Mongoose ODM, modelowanie relacji między użytkownikami a transakcjami

  • Mongoose ODM & Schema Design
  • User-Transaction Relationships
  • Data Indexing & Performance
  • Backup & Data Security

Hosting aplikacji na platformie Hostinger z konfiguracją środowiska produkcyjnego i zmiennymi środowiskowymi
