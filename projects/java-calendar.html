Java Calendar Application

Zaawansowana aplikacja desktopowa kalendarza stworzona w JavaFX z pełnym zarządzaniem wydarzeniami, przypomnieniami i persistentnym przechowywaniem danych

Zobacz kod źródłowy Dokumentacja

O Projekcie

Java Calendar Application to kompleksowa aplikacja desktopowa stworzona w języku Java z wykorzystaniem biblioteki JavaFX do budowy nowoczesnego interfejsu użytkownika. Projekt demonstruje zaawansowane techniki programowania obiektowego oraz zarządzania danymi w aplikacjach desktop.

Aplikacja oferuje pełnowartościowy system kalendarza z możliwością zarządzania wydarzeniami, ustawiania przypomnień i persistentnego przechowywania danych. Interface użytkownika został zaprojektowany zgodnie z zasadami Material Design i jest intuicyjny w obsłudze.

Technologie Użyte

Java 11+ JavaFX FXML CSS Styling File I/O Collections Framework OOP Design Patterns MVC Architecture

Kluczowe Funkcjonalności

  • Tworzenie i edycja wydarzeń kalendarzowych
  • System kategorii z kolorowym oznaczeniem
  • Przypomnienia z konfigurowalnymi alertami
  • Widoki: miesięczny, tygodniowy, dzienny
  • Persistentne przechowywanie w plikach
  • Eksport wydarzeń do CSV
  • Wyszukiwanie i filtrowanie wydarzeń
  • Responsywny interface JavaFX

Architektura Aplikacji

Projekt wykorzystuje wzorzec MVC (Model-View-Controller) z JavaFX jako warstwą prezentacji. Kontrolery zarządzają logiką biznesową, podczas gdy widoki FXML definiują strukturę interfejsu. Dane są serializowane i przechowywane lokalnie z wykorzystaniem Java I/O operations.

Komponenty Systemu

JavaFX Calendar View

Nowoczesny interfejs kalendarza z nawigacją między miesiącami i latami, stylowany CSS

Event Management System

Tworzenie, edycja i usuwanie wydarzeń z formularzami FXML i walidacją danych

Alert & Reminder System

Elastyczny system przypomnień z JavaFX Alert API i konfigurowalnymi powiadomieniami

Data Persistence Layer

Serializacja obiektów Java z automatycznym backup i odczytem z plików

Szczegóły Techniczne

JavaFX Architecture

Wykorzystanie FXML do separacji logiki od interfejsu, CSS do stylowania komponentów oraz kontrolerów do zarządzania interakcjami

Design Patterns

Implementacja wzorców MVC, Observer, Singleton i Factory dla modularnej i łatwo testowalnej architektury

Data Management

Wykorzystanie Java Collections Framework, File I/O operations i Object Serialization dla efektywnego zarządzania danymi

User Experience

Responsywny design z JavaFX binding, smooth animations i intuitive controls zgodne z JavaFX design guidelines

Kluczowe Implementacje

FXML Controllers

Implementacja kontrolerów JavaFX z @FXML annotations do zarządzania komponentami UI i event handling

CSS Styling

Custom style sheets dla JavaFX komponentów z modern flat design i responsywnymi kolorami

Data Persistence

Wykorzystanie ObjectOutputStream/ObjectInputStream do serializacji i deserializacji wydarzeń kalendarza

Alert System

JavaFX Alert API z custom dialogs do wyświetlania przypomnień i powiadomień użytkownika
